## 핵심 결론

연금계좌에서 인출하면 **법적으로 자동 순서**가 정해져 있어 사용자가 선택할 수 없습니다.

```
1순위: 비과세 재원
  - 세액공제 안 받은 납입 원금
  - ISA에서 이전한 미공제분
  → 인출 시 세금 0원, 한도 1500만원에도 미포함, 건보료 부과 X

2순위: 이연 퇴직소득
  - 회사 퇴직금이 IRP로 이전된 부분
  → 퇴직소득세 30~40% 감면 후 인출

3순위: 연금소득 재원 (합산 관리)
  - 세액공제 받은 납입 원금
  - ISA 이전 공제분 (Max 300만원/년)
  - 운용수익 전체
  → 연 1500만원 한도 내: 저율 분리과세 (3.3~5.5%)
  → 1500만원 초과: 종합과세 또는 16.5% 분리 선택
```

## 시각적 이해

이미지로 외우면 좋습니다 (출처: 사용자 제공 자료):

> 연금계좌 = 통합 물탱크. 위에서부터 1순위 → 3순위 순으로 빠짐.

## 단계별 인출 효과

### 1단계: 비과세 재원 (세금 0원)

이 재원을 **먼저 활용**하면 은퇴 초반의 세부담을 0에 가깝게 만들 수 있습니다.

<Callout tone="green">
**ISA → 연저펀 이전 전략의 핵심**: 3년마다 ISA 만기를 활용해 비과세 재원을 늘려두면, 은퇴 초반 세금 0원 인출 가능 기간이 길어집니다.
</Callout>

### 2단계: 이연 퇴직소득

퇴직금을 IRP로 받은 경우, 인출 시 **퇴직소득세의 30~40%만** 부담.
퇴직 직후 일시금으로 받지 말고 IRP에 두고 60~80세에 분할 인출이 유리.

### 3단계: 연금소득 재원

가장 일반적인 인출 재원. 세금 효과가 핵심.

| 인출액 | 과세 방식 |
| --- | --- |
| 연 1,500만원 이하 | 자동 분리과세 (3.3~5.5%) |
| 1,500만원 초과 | ① 16.5% 분리 또는 ② 종합과세 (사용자 선택) |
